# PTT-Shopping_Comparison
這是 2022 年在彰師修資料探勘的小作業!! :DD

## Problem to Solve
有許多人會利用PTT來販售二手商品，但由於PTT本身並非拍賣專用，故搜尋以及比價需耗費大量時間。
引此我們利用本學期所學習的網路爬蟲(BeautifulSoup)來製作簡易比價程式。

## How to Use?
### 1. 輸入想要搜尋的商品項
有iPad、iPhone、AirPods三種商品選項，輸入1、2、3來選取搜尋項

### 2. 成果
#### 有正常價錢標示項目
因PTT板內有些拍賣項目並無依照規格標示價錢，因此我們將有標示價錢者由低至高繪製成表格並印出拍賣者、貼文標題、貼文時間、價格以及原貼文網址

![圖片1](https://github.com/tsaiet/PTT-Shopping_Comparison/assets/67772010/5b7316bd-b8c9-4f62-b5c6-cb10702a9cb5)

#### 無價錢項目
無依照規格標示價錢者，我們將結果拍賣者、貼文標題、貼文時間、價格以及原貼文網址直接印出
